## geodjango
geodjango实现附近的人　　

说明：<https://blog.itswcg.com/2019-02/geodjango.html>

## Usage
```bash
$ git clone https://github.com/itswcg/geodjango
$ cd geodjango
$ python3 -m venv venv
$ source ./venv/bin/activate
$ pip install -r requirements.txt

# 更改设置postgresql数据库设置
# 更改设置中GEOS_LIBRARY_PATH,GDAL_LIBRARY_PATH路径设置
```
